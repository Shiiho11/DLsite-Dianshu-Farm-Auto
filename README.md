# DLsite-Dianshu-Farm-Auto
DLsite Dianshu Farm Auto is a Chrome extension that automatically performs the daily check-in on dianshu.jp to draw DLsite points.

DLsite Dianshu Farm Auto 是一个Chrome扩展，自动在dianshu.jp上每日签到，抽取DLsite点数。

## Usage

1. Install this extension in the Chrome browser or a browser that supports Chrome extensions. You may need to enable developer mode.
2. Open [dianshu.jp/farm](https://dianshu.jp/farm) and log in with your DLsite account.

The extension will perform daily check-in to draw points every time the browser is launched.

## 使用方法

1. 在Chrome浏览器或支持Chrome扩展的浏览器安装此扩展程序。可能需要打开开发者模式。
2. 打开[dianshu.jp/farm](https://dianshu.jp/farm)，登陆你的DLsite账号。

扩展会在每次启动浏览器时，进行每日签到抽取点数。
